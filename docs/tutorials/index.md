# Tutorials

Learn Whisker step by step with our tutorial series.

## Beginner Track

Start here if you're new to Whisker or interactive fiction.

1. [Your First Story](/tutorials/beginner/01-first-story) - Create a simple story
2. [Adding Choices](/tutorials/beginner/02-adding-choices) - Make your story interactive
3. [Using Variables](/tutorials/beginner/03-using-variables) - Track state and data
4. [Conditional Content](/tutorials/beginner/04-conditional-content) - Show content based on conditions
5. [Linking Passages](/tutorials/beginner/05-linking-passages) - Connect your story together

## Intermediate Track

Ready for more? Learn advanced techniques.

1. [Advanced Flow Control](/tutorials/intermediate/01-advanced-flow) - Gathers and tunnels
2. [Data Structures](/tutorials/intermediate/02-data-structures) - Lists, arrays, and maps
3. [Functions](/tutorials/intermediate/03-functions) - Reusable code blocks
4. [Styling](/tutorials/intermediate/04-styling) - Customize appearance
5. [Adding Media](/tutorials/intermediate/05-media) - Images, audio, and more

## Prerequisites

- A text editor (or use the [web editor](https://whisker.dev))
- Basic familiarity with interactive fiction concepts
