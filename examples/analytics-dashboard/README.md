# Analytics Dashboard Example

Real-time analytics viewer for tracking Whisker story performance using React and Recharts.

## Features

- ✅ Session metrics (total, completion rate, duration)
- ✅ Passage analytics (views, time spent)
- ✅ Interactive charts (Bar, Line)
- ✅ Responsive design
- ✅ Dark mode support
- ✅ Demo data generation

## Quick Start

```bash
npm install
npm run dev
```

## Stack

- React 18
- Recharts for data visualization
- Vite for build tooling
- @writewhisker/analytics for metrics
