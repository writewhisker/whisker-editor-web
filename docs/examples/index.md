## Running Examples

### Web Editor

Copy any example and paste it into the [web editor](https://whisker.dev).

### CLI

```bash
# Download an example
curl -O https://raw.githubusercontent.com/writewhisker/examples/main/basic/hello-world.ws

# Play it
whisker play hello-world.ws
```

### From Source

```bash
git clone https://github.com/writewhisker/examples.git
cd examples
whisker play basic/hello-world.ws
```
