# Getting Started

Welcome to Whisker! This guide will help you get up and running.

## What is Whisker?

Whisker is a language and toolset for creating interactive fiction - stories where readers make choices that affect the narrative. It's designed to be:

- **Easy to Learn**: Clean syntax that feels natural to write
- **Powerful**: Variables, conditions, and data structures for complex stories
- **Portable**: Run your stories on web, desktop, or mobile

## Quick Install

### Web Editor

Visit [whisker.dev](https://whisker.dev) to start writing immediately.

### CLI Tools

```bash
npm install -g @writewhisker/cli
whisker play story.ws
```

### As a Library

```bash
npm install @writewhisker/parser @writewhisker/story-player
```
