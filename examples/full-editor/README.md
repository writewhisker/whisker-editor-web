# Full Editor Example

Complete story authoring environment using Whisker's full editor-base package.

## Features

- ✅ All Whisker packages included via editor-base
- ✅ Complete editing environment
- ✅ Story metadata editing
- ✅ Passage management
- ✅ Auto-save functionality
- ✅ Export to JSON
- ✅ Responsive design

## Quick Start

```bash
npm install
npm run dev
```

## Usage

This example demonstrates using `@writewhisker/editor-base` which includes all Whisker packages:
- core-ts
- storage
- import/export
- analytics
- audio
- scripting
- github
- publishing
