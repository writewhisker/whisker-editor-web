## Running Showcase Examples

```bash
# Clone the examples repository
git clone https://github.com/writewhisker/examples.git

# Play the mystery game
whisker play examples/showcase/mystery/story.ws

# Export to playable HTML
whisker export examples/showcase/mystery/story.ws -o mystery.html
```
