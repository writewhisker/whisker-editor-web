# Beginner Tutorials

Start your Whisker journey here!

## Tutorials

1. [Your First Story](./01-first-story) - Create a simple story file
2. [Adding Choices](./02-adding-choices) - Make your story interactive
3. [Using Variables](./03-using-variables) - Track state and data
4. [Conditional Content](./04-conditional-content) - Show content based on conditions
5. [Linking Passages](./05-linking-passages) - Connect your story together

## Prerequisites

- A text editor or the [web editor](https://whisker.dev)
- About 30 minutes of time

## What You'll Learn

By the end of this track, you'll be able to:

- Create complete interactive stories
- Use variables to track player choices
- Show different content based on conditions
- Structure complex branching narratives
